# Experience Cloud-Integrationen

Die Adobe Experience Cloud ist ein umfassender Satz erstklassiger integrierter Lösungen, die auf einer gemeinsamen Datenplattform basieren und über einen gemeinsamen Satz an leistungsstarken Hauptdiensten verfügen.

## Hauptdienste - Aktivieren Ihrer Lösungen für Hauptdienste {#section_A3D024994DA3492F8435CFCC4EF035C2}

Hilfe: [Aktivieren Ihrer Lösungen für Hauptdienste](core-services/core-services.md#concept_07ED1D5C64234E77976E6D572E78FB9C)

Beschreibt Folgendes:

* Richten Sie Ihr Unternehmen in der Experience Cloud ein.
* Erhalten von Administratorrechten
* [Implementieren Sie den ID-Dienst der Experience Cloud](https://marketing.adobe.com/resources/help/en_US/mcvid/).
* Modernisieren von Analytics- und Target-Implementierungen über DTM
* Einführung in die Hauptdienste.

Lösungen oder Dienste:

* Activation - Dynamic Tag Management oder Experience Platform Launch
* Analytics
* Target
* [Experience Cloud ID-Dienst](https://marketing.adobe.com/resources/help/en_US/mcvid/)

## Experience Cloud ID-Dienst {#section_6ECCCFA2D84D4D4F88C879C799CA9D78}

Help: [Experience Cloud ID Service](https://marketing.adobe.com/resources/help/en_US/mcvid/)

Der ID-Dienst bietet eine universelle, beständige ID zum Identifizieren Ihrer Besucher über alle Experience Cloud-Lösungen hinweg. Er kann den generierten ID-Code für Dienste wie Analytics, Audience Manager, Target, Pulsmessung für Video und andere Experience Cloud-Lösungen oder Funktionen ersetzen.

**Anwendbare Lösungen oder Dienste**

* [Analytics](https://marketing.adobe.com/resources/help/en_US/mcvid/mcvid-setup-analytics.html)
* [Target](https://marketing.adobe.com/resources/help/en_US/mcvid/mcvid-setup-target.html)
* [Data Workbench](https://marketing.adobe.com/resources/help/en_US/mcvid/mcvid-dwb.html)

## Zielgruppen{#section_5F60D7B0833348B9A1D74663AADCB42C}

Hilfe: [Zielgruppen](audience-library/audience-library.md#topic_679810123CAA4E0CA4FA3417FB0100C7)

Erstellen und verwalten Sie Zielgruppen in der Experience Cloud-Zielgruppenbibliothek. Zielgruppen können aus verschiedenen Quellen erstellt bzw. abgeleitet werden:

* In der [!DNL Experience Cloud] neu erstellt
* Aus [!DNL Analytics]-Segmenten, die in der [!DNL Experience Cloud] veröffentlicht werden
* Von [!DNL Audience Manager].

**Anwendbare Lösungen oder Dienste**

* [Aktivitäten](https://marketing.adobe.com/resources/help/en_US/target/target/c_activities.html) in Adobe Target
* [Segmentierung](https://marketing.adobe.com/resources/help/en_US/analytics/audiences/aam-analytics-segments.html) in Audience Manager
* [Media Optimizer](https://enterprise.efrontier.com/CMDashboard/help/internal/concepts_and_features/media_optimizer_integration_with_adobe_marketing_cloud.htm) (Anmeldung erforderlich)

## Kundenattribute {#section_6A9EA6847F654F129381869E5016626C}

Hilfe: [Kundenattribute](attributes/attributes.md#concept_ACFEE7C8B8E94875BA0825CDF4913AF1)

## Advertising Cloud {#section_9B1935F8BBC147C89C6DB68A35CB1BAB}

Help (login required): [Integration with Adobe Experience Cloud Solutions and Core Services](https://enterprise.efrontier.com/CMDashboard/help/internal/concepts_and_features/media_optimizer_integration_with_adobe_marketing_cloud.htm)

**Anwendbare Lösungen oder Dienste**

**Analytics:** kann Daten zu Site-Interaktion und Konversion täglich an Media Optimizer senden, wo die Daten zur Anzeigenoptimierung und Berichterstellung zur Verfügung stehen. Außerdem kann Media Optimizer Daten zu Suchmaschinen und Traffic-Daten in sozialen Netzwerken täglich an Analytics senden, wo diese Daten dann für Berichte in Reports &amp; Analysen, Report Builder und Ad-hoc-Analysefunktionen verfügbar sind.

**Dynamischer Tag-Manager:** Mit dem [dynamischen Tag-Manager können Sie Media Optimizer-Tags zur Konversionsverfolgung auf Pixel-Basis](https://marketing.adobe.com/resources/help/en_US/dtm/t_media_opt_rule.html) sowie Verfolgungs-Tags für Drittanbieteranwendungen erstellen, und zwar für Ihre Landingpages für Suchvorgänge, soziale Netzwerke und Display-Anzeigen. (Sie können Media Optimizer-Tags auch direkt in Media Optimizer erstellen.)

**Experience Cloud-Zielgruppen:** (Werbetreibende mit Display-Verwaltung) Sie können beliebige Ihrer [Adobe Experience Cloud-Zielgruppen](https://marketing.adobe.com/resources/help/en_US/mcloud/audience_library.html) als Zielgruppen für Ihre Display-Anzeigen verwenden. Sie können Zielgruppen automatisch verwenden, die Sie in Adobe Experience Cloud erstellt haben, und Zielgruppen aus Analytics, die Sie in Adobe Experience Cloud veröffentlicht haben. Sie können Zielgruppen aus Audience Manager verwenden, wenn das Media Optimizer-Konto mit den entsprechenden Berechtigungen konfiguriert wurde. Wenden Sie sich an Ihren Account Manager, wenn Sie weitere Informationen über den Zugriff auf die Adobe Experience Cloud, die dazugehörigen Hauptdienste „Profiles &amp; Audiences“ sowie über die Ersteinrichtung von Media Optimizer- und Adobe Experience Cloud-Zielgruppen erfahren möchten. **Hinweis:** Alle Zielgruppen, die Sie in Adobe Experience Cloud veröffentlicht haben, stehen ebenfalls für Aktivitäten in Adobe Target zur Verfügung, falls Sie Target verwenden.

**Experience Cloud Assets:** (Werbetreibende mit Display-Verwaltung) Sie können beliebige Ihrer Adobe Experience Cloud-Assets als Grundlage für Ihre Display-Anzeigen nutzen, wenn Sie die neue Display-Beta-Ansicht verwenden. Sie müssen [über Adobe Experience Cloud bei Media Optimizer angemeldet sein](https://enterprise-test.efrontier.com/CMDashboard/help/internal/getting_started/t_log_in_from_adobe_marketing_cloud.htm), um auf Ihre Adobe Experience Cloud-Assets zuzugreifen. Wenden Sie sich an Ihren Account Manager, um Informationen über den Zugriff auf die Adobe Experience Cloud zu erhalten.

**Experience Cloud-Benachrichtigungen:** Über den Benachrichtigungslink im oberen Bereich jeder Seite können Sie alle Warnhinweise, die von Ihren Beta-Warnhinweisvorlagen für Suchen generiert wurden, sowie Systemaktualisierungen, Posts, Erwähnungen und gemeinsam genutzte Assets in Bezug auf die Adobe Experience Cloud anzeigen. Sie müssen [über die Adobe Experience Cloud bei Media Optimizer angemeldet sein](https://enterprise-test.efrontier.com/CMDashboard/help/internal/getting_started/t_log_in_from_adobe_marketing_cloud.htm), um auf Ihre Benachrichtigungen zuzugreifen. Wenden Sie sich an Ihren Account Manager, um Informationen über den Zugriff auf die Adobe Experience Cloud zu erhalten.
