# Experience Cloud-Integrationen

Die Adobe Experience Cloud ist ein umfassender Satz erstklassiger integrierter Lösungen, die auf einer gemeinsamen Datenplattform basieren und über einen gemeinsamen Satz an leistungsstarken Hauptdiensten verfügen.

## Hauptdienste - Aktivieren Ihrer Lösungen für Hauptdienste {#section_A3D024994DA3492F8435CFCC4EF035C2}

Hilfe: [Aktivieren Ihrer Lösungen für Hauptdienste](core-services/core-services.md#concept_07ED1D5C64234E77976E6D572E78FB9C)

Beschreibt Folgendes:

* Richten Sie Ihr Unternehmen in der Experience Cloud ein.
* Erhalten von Administratorrechten
* [Implementieren Sie den Experience Cloud ID-Dienst](https://marketing.adobe.com/resources/help/de_DE/mcvid/).
* Modernisieren von Analytics- und Target-Implementierungen über DTM
* Einführung in die Hauptdienste

Lösungen oder Dienste:

* Activation - Dynamic Tag Management oder Experience Platform Launch
* Analytics
* Target
* [Experience Cloud ID-Dienst](https://marketing.adobe.com/resources/help/de_DE/mcvid/)

## Experience Cloud ID-Dienst {#section_6ECCCFA2D84D4D4F88C879C799CA9D78}

Hilfe: [Experience Cloud ID-Dienst](https://marketing.adobe.com/resources/help/de_DE/mcvid/)

Der ID-Dienst bietet eine universelle, beständige ID zum Identifizieren Ihrer Besucher über alle Experience Cloud-Lösungen hinweg. Er kann den generierten ID-Code für Dienste wie Analytics, Audience Manager, Target, Pulsmessung für Video und andere Experience Cloud-Lösungen oder Funktionen ersetzen.

**Anwendbare Lösungen oder Dienste**

* [Analytics](https://marketing.adobe.com/resources/help/de_DE/mcvid/?f=mcvid-setup-analytics)
* [Target](https://marketing.adobe.com/resources/help/de_DE/mcvid/?f=mcvid-setup-target)
* [Data Workbench](https://marketing.adobe.com/resources/help/de_DE/mcvid/?f=mcvid-dwb)

## Zielgruppen{#section_5F60D7B0833348B9A1D74663AADCB42C}

Hilfe: [Zielgruppen](audience-library/audience-library.md#topic_679810123CAA4E0CA4FA3417FB0100C7)

* In der [!DNL Experience Cloud] neu erstellt

* Von [!DNL Audience Manager].

**Anwendbare Lösungen oder Dienste**

## Advertising Cloud {#section_9B1935F8BBC147C89C6DB68A35CB1BAB}

Hilfe (Anmeldung erforderlich): [Integration mit Adobe Experience Cloud-Lösungen und Hauptdiensten](https://enterprise.efrontier.com/CMDashboard/help/internal/concepts_and_features/media_optimizer_integration_with_adobe_marketing_cloud.htm)

**Anwendbare Lösungen oder Dienste**

**Analytics:** kann Daten zu Site-Interaktion und Konversion täglich an Media Optimizer senden, wo die Daten zur Anzeigenoptimierung und Berichterstellung zur Verfügung stehen. Außerdem kann Media Optimizer Daten zu Suchmaschinen und Traffic-Daten in sozialen Netzwerken täglich an Analytics senden, wo diese Daten dann für Berichte in Reports &amp; Analysen, Report Builder und Ad-hoc-Analysefunktionen verfügbar sind.

**Dynamischer Tag-Manager:** Sie können den [dynamischen Tag-Manager verwenden, um pixelbasierte Konvertierungs-Tracking-Tags von Media Optimizer](https://marketing.adobe.com/resources/help/de_DE/dtm/?f=t_media_opt_rule) sowie Tracking-Tags von Drittanbietern für Landingpages von Such-, Social- und Display-Anzeigen zu erstellen. (Sie können Media Optimizer-Tags auch direkt in Media Optimizer erstellen.)

**Experience Cloud-Zielgruppen:** (Werbetreibende mit Display-Verwaltung) Sie können beliebige [Adobe Experience Cloud-Zielgruppen](https://marketing.adobe.com/resources/help/de_DE/mcloud/?f=audience_library) als Ziele für Ihre Display-Anzeigen verwenden. Sie können Zielgruppen automatisch verwenden, die Sie in Adobe Experience Cloud erstellt haben, und Zielgruppen aus Analytics, die Sie in Adobe Experience Cloud veröffentlicht haben. Sie können Zielgruppen aus Audience Manager verwenden, wenn das Media Optimizer-Konto mit den entsprechenden Berechtigungen konfiguriert wurde. Wenden Sie sich an Ihren Account Manager, wenn Sie weitere Informationen über den Zugriff auf die Adobe Experience Cloud, die dazugehörigen Hauptdienste „Profiles &amp; Audiences“ sowie über die Ersteinrichtung von Media Optimizer- und Adobe Experience Cloud-Zielgruppen erfahren möchten. **Hinweis:** Alle Zielgruppen, die Sie in Adobe Experience Cloud veröffentlicht haben, stehen ebenfalls für Aktivitäten in Adobe Target zur Verfügung, falls Sie Target verwenden.

**Experience Cloud Assets:** (Werbetreibende mit Display-Verwaltung) Sie können beliebige Ihrer Adobe Experience Cloud-Assets als Grundlage für Ihre Display-Anzeigen nutzen, wenn Sie die neue Display-Beta-Ansicht verwenden. Sie müssen [bei Media Optimizer über die Adobe Experience Cloud eingeloggt sein](https://enterprise-test.efrontier.com/CMDashboard/help/internal/getting_started/t_log_in_from_adobe_marketing_cloud.htm)), um auf Ihre Adobe Experience Cloud-Assets zugreifen zu können. Wenden Sie sich an Ihren Account Manager, um Informationen über den Zugriff auf die Adobe Experience Cloud zu erhalten.

**Experience Cloud-Benachrichtigungen:** Über den Benachrichtigungslink im oberen Bereich jeder Seite können Sie alle Warnhinweise, die von Ihren Beta-Warnhinweisvorlagen für Suchen generiert wurden, sowie Systemaktualisierungen, Posts, Erwähnungen und gemeinsam genutzte Assets in Bezug auf die Adobe Experience Cloud anzeigen. Sie müssen [bei Media Optimizer über die Adobe Experience Cloud eingeloggt sein](https://enterprise-test.efrontier.com/CMDashboard/help/internal/getting_started/t_log_in_from_adobe_marketing_cloud.htm)), um auf Ihre Benachrichtigungen zugreifen zu können. Wenden Sie sich an Ihren Account Manager, um Informationen über den Zugriff auf die Adobe Experience Cloud zu erhalten.
